# Istruzioni per installare ed eseguire il progetto React + Vite

Segui questi passaggi per installare ed eseguire il progetto sul tuo computer.

---

Apri il terminale e digita deve fare in command prompt:

1.
```
cd technical-writing-trello-frontend
```
2.
```
npm install
```
3.
```
npm run dev
```
